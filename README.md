# Voting App
##### By Stijn Servaes

### Built with
* Node JS
* Express JS
* Bootstrap 5.3

## Getting Started
### Install Instructions

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
3. Create a `.env` file or create the following process.env variables
   ```js
   ADMIN_USERNAME=
   ADMIN_PASSWORD=
   PORT=
   ```
4. Use the following command to start the server
   ```
   npm start
   ```
